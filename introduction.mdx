WiFi Hacking is an open-source Python script that bundles 18 wireless network security testing capabilities into a single interactive CLI menu. Built for security researchers and penetration testers, it wraps industry-standard Kali Linux tools — aircrack-ng, Reaver, Nmap, Hydra, Metasploit, and more — behind a unified interface so you can move quickly through a test engagement without switching between separate tools.

<Warning>
  This tool is intended exclusively for authorized penetration testing and educational research. Only use it against networks and systems you own or have explicit written permission to test. Unauthorized access to computer networks is illegal in most jurisdictions.
</Warning>

## What's included

WiFi Hacking integrates the following capabilities in one menu-driven script:

| # | Feature | Underlying tool |
|---|---------|-----------------|
| 1–2 | Start / stop wireless monitor mode | airmon-ng |
| 3 | Scan nearby wireless networks | airodump-ng |
| 4 | Capture WPA/WPA2 handshakes | airodump-ng + aireplay-ng |
| 5 | Install all required tools automatically | apt-get |
| 6–8 | Crack captured handshakes (rockyou, custom wordlist, or ESSID) | aircrack-ng |
| 9 | Generate custom wordlists | crunch |
| 10 | WPS brute-force attacks | reaver |
| 11 | Network host discovery | nmap |
| 12 | Run Metasploit exploits | msfconsole |
| 13 | SSH brute-force login | hydra |
| 14 | Evil-twin phishing attack | wifiphisher |
| 15 | Web server vulnerability scan | nikto |
| 16 | Port listener / reverse shell staging | netcat |
| 17 | Web directory enumeration | gobuster |
| 18 | SQL injection testing | sqlmap |

## Supported platforms

- Kali Linux (primary)
- BlackArch Linux
- Ubuntu
- Parrot OS
- Kali NetHunter
- Termux (rooted Android devices)
